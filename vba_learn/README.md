## 📘 学習内容の概要

### 基本文法・構文
- 変数の宣言（`Dim`）とスコープ
- 条件分岐：`If` / `ElseIf` / `Select Case`
- 繰り返し処理：`For` / `Do Until`

### Excel操作
- ワークシートの追加・削除・名前変更
- セル/行/列の選択・入力・書式設定
- 表の自動整形（テーブル化・罫線）

### その他処理
- メッセージボックス・入力ボックス
- ファイル／フォルダ操作（選択・保存・開く）
- 自動フィルタ・並べ替え・削除処理
